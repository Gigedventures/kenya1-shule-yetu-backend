# Shule Yetu Student Dashboards

Student dashboards are split by `SchoolLevel` and routed through `student_dashboard_router_screen.dart`.

- `primary/` for Grade 1-5
- `junior/` for Grade 6-8
- `senior/` for Grade 9-12
- `tertiary/` for college, TVET, and university

This layer stays mock-data driven for now; backend integration should plug into the same router later.
